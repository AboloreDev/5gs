5GS Photography Media Website
A Next.js-powered photography media platform where users can explore content, book appointments, rent photography equipment, and interact with an admin. This project is frontend-only, utilizing Context API, Formik & Yup, and styled with Tailwind CSS.

🚀 Features
User (Client) Side
✅ Landing Page – Engaging homepage showcasing photography services.
✅ Dashboard – Users can view admin posts, check available rentals, and interact.
✅ Book Appointments – Schedule photography sessions.
✅ Rent Page – Browse and "rent" available items.
✅ Messaging – Send messages to the admin.
✅ Notifications – Receive updates from the admin.
✅ Rate Experience – Provide feedback on the service.

Admin Side
✅ Login System (Frontend Only) – Simulated authentication.
✅ Upload Media – Add photos and videos.
✅ Manage Rentals – Upload and manage rentable photography items.
✅ Check Messages – View and respond to user inquiries.
✅ Approve Notifications – Manage user requests and updates.

🛠️ Tech Stack
Framework: Next.js
State Management: Context API
Forms & Validation: Formik & Yup
Styling: Tailwind CSS
